Foundation

The foundation document defines the conceptual structure of the research. It establishes the difference between relational color systems and algorithmic representations.

Core Problem

The research examines color as a system that cannot be reduced to independent variables. Instead, color is treated as relational, where each element depends on the entire structure.

Algorithmic systems attempt to isolate and control color, but fail to reproduce its generative behavior. This gap defines the central problem of the research.

Active Work

  • Relational structure analysis

  • Experimental construction of color systems
  • Documentation and structural notes

Experiments

Experiments are used to test how color behaves when treated as a system rather than a parameter. Both visual and structural approaches are used.

Results will appear as images, notes, and structured observations.
