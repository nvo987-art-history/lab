## Overview

NV0987 Lab is an independent experimental research environment focused on:

- Relational color systems  
- Perceptual development  
- AI image generation  
- Semantic web infrastructure  
- Decentralized identity (DID)  
- Machine-readable research systems  

The lab operates as a modular node within a distributed network where identity, knowledge, and research processes are structurally interconnected.

## Research Focus

The project investigates two fundamentally different modes of color formation:

### 1. Human perceptual color formation
- Relational  
- Temporal  
- Experience-based  

### 2. AI-generated color formation
- Computational  
- Instant  
- Statistical output  

The goal is not evaluation, but understanding the underlying generative logic of each system.
